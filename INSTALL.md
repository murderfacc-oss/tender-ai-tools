# Установка для партнёра

Получаешь от меня файл **`tender-ai.plugin`** — это всё, что нужно.

---

## 1. Установить плагин

1. Открой **Claude Code**.
2. Перетащи `tender-ai.plugin` прямо в окно Claude Code.
3. В появившемся диалоге нажми **Install**.

Плагин установлен. В нём уже подключены 3 скилла и MCP-сервер.

---

## 2. Установить Python-зависимости (один раз)

MCP-серверу нужен Python 3.10+ и несколько библиотек.

Открой терминал (на Windows: Win+R → `cmd` → Enter) и выполни:

```
pip install mcp requests beautifulsoup4 lxml python-docx
```

Если `pip` не находится — установи Python с python.org с галочкой
«Add Python to PATH» и повтори команду.

---

## 3. Перезапустить Claude Code

Закрой и снова открой Claude Code, чтобы подцепился MCP-сервер.

---

## Проверка

В новом чате:

- Напечатай `@zakupki-eis` — должен появиться список инструментов
  для скачивания документов с ЕИС.
- Прикрепи документ закупки и напиши «прочитай закупку» — должен
  сработать скилл `scan-zakupki`.

---

## Обновление

Когда я выпускаю новую версию — присылаю новый `tender-ai.plugin`.
Перетаскиваешь в Claude Code, **Install** — плагин обновляется.

`pip install` заново делать не нужно (если только в обновлении не
добавились новые библиотеки — я предупрежу).
