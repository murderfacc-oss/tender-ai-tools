# Установка для партнёра

Получаешь от меня файл **`tender-ai.plugin`** — это всё, что нужно.

---

## Шаги

1. **Открой Claude Code.**
2. **Перетащи** `tender-ai.plugin` прямо в окно Claude Code.
3. В появившемся диалоге нажми **Install**.
4. **Перезапусти** Claude Code.

Готово. При первом обращении к MCP-серверу он сам подтянет нужные
Python-библиотеки — никаких команд в терминале выполнять не нужно.

---

## Требования

- **Claude Code** (десктоп-приложение).
- **Python 3.10+** установлен в системе.
  Если нет — скачай с [python.org](https://python.org) с галочкой
  «Add Python to PATH».

---

## Проверка

В новом чате:

- Напиши `@zakupki-eis` — должен появиться список инструментов
  для скачивания документов с ЕИС.
- Прикрепи документ закупки и напиши **«прочитай закупку»** —
  должен сработать скилл `scan-zakupki`.

---

## Обновление

Когда я выпускаю новую версию — присылаю новый `tender-ai.plugin`.
Перетаскиваешь в Claude Code → **Install** → готово.
