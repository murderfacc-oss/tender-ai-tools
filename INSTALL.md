# Установка для партнёра

Раньше всё ставилось одним `.plugin` файлом — от этого подхода
отказались. Теперь скиллы и MCP-сервер ставятся **по отдельности** —
так проще обновлять и отлаживать.

Получаешь от меня:
- zip-архивы скиллов (`scan-zakupki-v*.zip`, `zhaloba-fas-v*.zip`, `zapros-razyasneniy-v*.zip`),
- папку `mcp/` (или ссылку на репо).

---

## 1. Скиллы

Каждый zip — отдельный скилл. Установи по очереди в Claude Code:

1. Открой Claude Code.
2. Перетащи zip-архив скилла в окно.
3. В появившемся диалоге нажми **Install**.
4. Повтори для остальных скиллов.
5. Перезапусти Claude Code.

---

## 2. MCP-сервер `zakupki-eis`

Положи папку `mcp/` в удобное место (рекомендую короткий ASCII-путь —
например, `C:\MCP\zakupki-eis\`).

Зарегистрируй MCP-сервер в Claude Code командой:

```bash
claude mcp add --scope user zakupki-eis python C:\MCP\zakupki-eis\launcher.py
```

(путь подставь свой). После — перезапусти Claude Code.

При первом обращении к MCP `launcher.py` сам поставит Python-зависимости
через pip — никаких ручных команд больше не нужно.

---

## Требования

- **Claude Code** (десктоп-приложение).
- **Python 3.10+** в системе.
  Если нет — скачай с [python.org](https://python.org) с галочкой
  «Add Python to PATH».

---

## Проверка

В новом чате:

- Напиши `@zakupki-eis` — должен появиться список инструментов
  для скачивания документов с ЕИС.
- Прикрепи документ закупки и напиши **«прочитай закупку»** —
  должен сработать скилл `scan-zakupki`.

---

## Обновление

- **Скилл изменился** — присылаю новый zip, перетаскиваешь в Claude Code → **Install**.
- **MCP изменился** — присылаю новые файлы в `mcp/`, ты их кладёшь поверх старых.
  Если поменялся список зависимостей — `launcher.py` сам поставит недостающее.
