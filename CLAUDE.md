# MCP-сервер «Закупки ЕИС» — контекст проекта

## Цель

MCP-сервер скачивает документы закупок с **zakupki.gov.ru** прямо в чат
Claude Code. Подключается один раз, работает бесплатно, без API-ключей
и регистраций.

## Владелец проекта

Артём — монтаж слаботочных систем (видеонаблюдение, СКУД, речевое оповещение, СКС).
Участвует в закупках по 44-ФЗ и 223-ФЗ.

## Стек

- Python 3.10+
- MCP SDK: `mcp` (FastMCP)
- HTTP: `requests`
- Парсинг страниц: `beautifulsoup4`, `lxml`
- Источник данных: zakupki.gov.ru (прямой скрап, без API)

## Инструменты MCP

| Инструмент | Что делает |
|---|---|
| `download_tender` | Скачать все документы извещения + (если есть) подписанный контракт по номеру извещения |
| `download_contract` | Скачать только документы контракта по реестровому номеру |
| `check_tender_updates` | Проверить, обновились ли документы извещения на сайте |
| `update_tender` | Скачать обновления и показать что изменилось |

Файлы автоматически раскладываются по подпапкам: `1_ТЗ`, `2_Контракт`,
`3_НМЦК`, `4_Смета`, `5_Документация`, `6_Протоколы`, `7_Прочее`.

## Подключение к Claude Code

Через `claude mcp add --scope user zakupki-eis python /полный/путь/server.py`,
либо через UI: Settings → MCP Servers → Add server → stdio →
Command `python`, Args `<путь>/server.py`.

После — перезапуск Claude Code, проверить через `@zakupki-eis` в новом чате.
