# CHANGELOG — zapros-razyasneniy

## v0.1.1 (2026-05-02) — две правки

**Правка 1: все вопросы — в один документ.**
Раньше каждый сюжет шёл отдельным запросом → быстро тратил лимит (3 запроса для электронного аукциона). Теперь все вопросы по закупке упаковываются в один файл — пронумерованные блоки внутри одного документа. Один файл = один запрос на счётчике.

**Правка 2: убраны реквизиты сторон из файла.**
Раньше в шапке были адресат (заказчик) и заявитель (ИНН, КПП, адрес, контакты). Теперь нет — площадка ЕИС автоматически прикрепляет эти данные при подаче запроса. В Word — только заголовок, обращение, тело, подпись, дата.

**Изменено:** `SKILL.md`, `references/tipy-voprosov.md`, `references/strategiya.md`, `references/chek-list.md`, `references/shablon-word.md`, `references/generate_request.py`. JSON-схема входных данных упрощена (больше нет полей `zakazchik`, `zayavitel`).

## v0.1 (2026-05-02) — инициал

**Покрытие:**
- 44-ФЗ, электронный аукцион + открытый конкурс (там, где разъяснения предусмотрены)
- Пять стратегических сюжетов: эквивалент, коллизия, неоднозначность, объём, обоснование

**Структура:**
- `SKILL.md` — точка входа, триггеры, workflow
- `references/tipy-voprosov.md` — 5 сюжетов с готовыми шаблонами
- `references/chelovecheskiy-yazyk.md` — таблица перевода технического жаргона
- `references/strategiya.md` — когда задавать вопрос, когда воздержаться
- `references/chek-list.md` — финальная проверка перед отправкой
- `references/shablon-word.md` — структура .docx-документа
- `references/generate_request.py` — скрипт на python-docx для генерации файла

**Главный принцип:**
Запрос читает специалист по закупкам, не технарь. Текст пишется обывательским языком, без жаргона и без юридических лекций.

**Не покрыто (отложено):**
- Запросы по 223-ФЗ — у каждого заказчика свой регламент
- Письма заказчику при исполнении (эквивалент, продление срока) — другой жанр
- Протоколы разногласий при подписании контракта (для котировок) — отдельный документ
- Запрос котировок не имеет процедуры разъяснений (ст. 50 44-ФЗ)
