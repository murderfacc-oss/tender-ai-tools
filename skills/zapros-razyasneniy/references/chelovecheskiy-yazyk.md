# Перевод технического языка в обывательский

Главное: запрос читает специалист по закупкам школы или садика. Не технарь.

Если он не поймёт термин с первого взгляда — ответ будет либо формальным, либо не в нашу пользу. Поэтому каждый технический термин в запросе **обязательно** заменяем на бытовое описание.

---

## Видеонаблюдение

| Технический термин | Обывательское объяснение |
|---|---|
| WDR (Wide Dynamic Range) | возможность качественно снимать на свету и в тени одновременно |
| DWDR | то же самое, но программный (более слабый) вариант |
| Lux (чувствительность) | способность камеры снимать в темноте |
| ИК-подсветка | встроенная подсветка для съёмки в полной темноте |
| Битрейт | качество записи, влияющее на размер файла |
| FPS / частота кадров | плавность видео (сколько кадров в секунду записывается) |
| Видеоаналитика | автоматическое распознавание событий — например, движения или лиц |
| AI-аналитика | то же, через искусственный интеллект |
| H.264 / H.265 | способ сжатия видео для уменьшения размера записи |
| ONVIF | стандарт совместимости камер разных производителей |
| IP-камера | камера, подключаемая по сетевому кабелю |
| Аналоговая камера | камера, подключаемая по коаксиальному кабелю |
| HD / Full HD / 4K | качество изображения (от низкого к высокому) |
| 2 МП / 4 МП / 8 МП | количество точек на снимке (чем больше, тем чётче) |
| IP66 / IP67 | защита от пыли и воды (для уличных камер) |
| Антивандальный корпус | прочный корпус, защищающий от ударов |
| PTZ / поворотная камера | камера, которая может поворачиваться и приближать изображение |

---

## Сети и питание

| Технический термин | Обывательское объяснение |
|---|---|
| PoE (Power over Ethernet) | питание камеры по тому же сетевому кабелю |
| PoE+ / PoE++ | то же, но более мощное питание |
| Коммутатор | устройство, к которому подключаются все камеры одной сетью |
| Управляемый коммутатор | коммутатор с возможностью настройки |
| Витая пара / UTP | сетевой кабель |
| Кабель категории 5e / 6 | разные классы качества сетевого кабеля |
| Оптоволокно | специальный кабель для передачи сигнала на большие расстояния |
| Источник бесперебойного питания (ИБП) | устройство, которое поддерживает работу при отключении света |
| VLAN | разделение сети на части |
| RJ-45 | стандартный сетевой разъём |

---

## Регистратор и архив

| Технический термин | Обывательское объяснение |
|---|---|
| Видеорегистратор / NVR / DVR | устройство для записи и хранения видео |
| Каналы регистратора | количество камер, которые можно к нему подключить |
| RAID | способ объединения нескольких дисков для надёжности |
| HDD | жёсткий диск для хранения записей |
| Глубина архива | сколько суток будут храниться записи |
| Объём хранения / ёмкость архива | сколько места занимают записи (в терабайтах) |
| Транскодирование | обработка видео для одновременного просмотра разных потоков |

---

## СКУД (контроль доступа)

| Технический термин | Обывательское объяснение |
|---|---|
| Контроллер СКУД | основное устройство, управляющее системой доступа |
| Считыватель | устройство, к которому подносят карту или браслет |
| Электромагнитный замок | замок, удерживаемый магнитом |
| Электромеханический замок | замок с механическим открыванием |
| Доводчик | устройство, плавно закрывающее дверь |
| RFID-карта | бесконтактная карта доступа |
| Mifare | стандарт защищённых карт |
| Wiegand / OSDP | стандарты подключения считывателей |
| Турникет | устройство для прохода по одному человеку |

---

## Пожарная сигнализация (АПС / СОУЭ)

| Технический термин | Обывательское объяснение |
|---|---|
| АПС | автоматическая пожарная сигнализация |
| СОУЭ | система оповещения и управления эвакуацией при пожаре |
| Извещатель / датчик | устройство, обнаруживающее пожар |
| Дымовой / тепловой / ручной извещатель | разные типы датчиков пожара |
| ППК (приёмно-контрольный прибор) | основной прибор управления сигнализацией |
| Адресный / неадресный | с возможностью определить конкретный сработавший датчик / без |
| Звуковой оповещатель | громкоговоритель для оповещения о пожаре |
| Световой оповещатель | мигающий указатель «Выход» или «Пожар» |
| Шлейф | линия проводов, соединяющая датчики с прибором |
| Огнестойкий кабель | кабель, сохраняющий работоспособность при пожаре |

## Юридические термины — тоже упрощаем

| Полная форма | Упрощённая |
|---|---|
| В соответствии с п. 1 ч. 1 ст. 33 Закона № 44-ФЗ | согласно пункту 1 части 1 статьи 33 закона о госзакупках |
| Описание объекта закупки | описание того, что заказчик покупает |
| Эквивалент | замена на похожий товар другого производителя |
| Совокупность характеристик | все требования вместе |
| Нацрежим | правила покупки российских товаров |
| Реестр Минпромторга | список российской промышленной продукции |
| Допсоглашение | дополнение к контракту, изменяющее его условия |
| КС-2 / КС-3 | акты выполненных работ и справки о стоимости |
| Мотивированный отказ | официальный отказ заказчика принять работу с указанием причин |

В тексте запроса юридические термины используем только при крайней необходимости (например, «часть 1 статьи 33 Закона о контрактной системе» в запросе сюжета 5 — там без неё не обойтись). Везде, где можно — на простой язык.

---

## Тест на понятность

Перед отправкой запроса — прочитать его вслух и спросить себя:
**«Если бы это читал бухгалтер школы, который вообще не разбирается в технике — он понял бы, в чём именно вопрос?»**

Если хоть в одной фразе ответ «не уверен» — переписываем эту фразу.

## Примеры до/после

### Пример 1

**До:**
> «Согласно п. 4.2 ТЗ камера должна обладать функцией WDR не менее 120 dB и поддерживать кодек H.265 с переменным битрейтом до 8 Мбит/с при разрешении 5MP.»

**После:**
> «В пункте 4.2 Технического задания указаны характеристики камеры: возможность качественно снимать на свету и в тени одновременно, способ сжатия видео и качество записи. Конкретные значения этих параметров — [перечислить простым языком].»

---

### Пример 2

**До:**
> «Допустимо ли применение IP-камер с PoE+ питанием при условии соблюдения требований по WDR, локализации шумов и времени экспозиции?»

**После:**
> «Допустимо ли применение камер, которые подключаются по сетевому кабелю и получают по нему же питание, при условии, что все указанные в Техническом задании характеристики выполняются не хуже, чем у конкретной модели в смете?»

---

### Пример 3

**До:**
> «Какова требуемая ёмкость архива системы видеонаблюдения с учётом FPS и битрейта при глубине 30 суток?»

**После:**
> «Какой объём дискового пространства считается обязательным для системы видеонаблюдения, чтобы записи сохранялись не менее 30 суток? Прошу указать конкретное значение в терабайтах.»
