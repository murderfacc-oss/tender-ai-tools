# MCP-сервер «Закупки ЕИС» — контекст проекта

## Цель

MCP-сервер скачивает документы закупок прямо в чат Claude Code через
**ГосПлан API v2** (`https://v2.gosplan.info`). Подключается один раз,
работает без регистрации. До 2026-07-01 — без API-ключей; после введения
ключей достаточно задать переменную окружения `GOSPLAN_API_KEY`.

Прямой скрап zakupki.gov.ru удалён в v0.7.0 (миграция задокументирована в
`docs/superpowers/specs/2026-05-18-gosplan-api-migration-design.md`).

## Владелец проекта

Артём — монтаж слаботочных систем (видеонаблюдение, СКУД, речевое оповещение, СКС).
Участвует в закупках по 44-ФЗ и 223-ФЗ.

## Стек

- Python 3.10+
- MCP SDK: `mcp` (FastMCP)
- HTTP: `requests`
- Источник данных: ГосПлан API v2 (`https://v2.gosplan.info`)

## Инструменты MCP

| Инструмент | Что делает |
|---|---|
| `download_tender` | Скачать документы извещения закупки по номеру (44-ФЗ или 223-ФЗ — определяется автоматически). Сохраняет `print_form.txt` (текстовая сводка) и раскладывает файлы по подпапкам |
| `download_contract` | Скачать документы подписанного контракта в подпапку `Контракт/`. Использует эндпоинты `/contracts/{n}` (сам договор) и `/contracts/{n}/procedures` (КС-2/КС-3/приёмка/платёжки/экспертиза). Передаётся реестровый номер КОНТРАКТА, не извещения |
| `check_tender_updates` | Проверить, обновились ли документы извещения (без скачивания) |
| `update_tender` | Скачать обновления и показать что изменилось |

Файлы автоматически раскладываются по подпапкам: `1_ТЗ`, `2_Контракт`,
`3_НМЦК`, `4_Смета`, `5_Документация`, `6_Протоколы`, `7_Прочее`.

**Текстовая сводка извещения** сохраняется в корне папки закупки:
- `print_form.txt` — чистый текст (для анализа в LLM, построен из JSON
  ответа ГосПлан API, без HTML-разметки)

Это сводная карточка по извещению с заказчиком, объектом закупки,
требованиями, графиком — часто полнее, чем разрозненные приложения.

## Структура проекта

```
mcp/
├── CLAUDE.md               # этот файл
├── server.py               # MCP-сервер (FastMCP, 4 инструмента)
├── gosplan_client.py       # HTTP-клиент ГосПлан API v2
├── gosplan_extract.py      # извлечение вложений и текстовой сводки из JSON
├── gosplan_store.py        # раскладка файлов, метаданные, diff обновлений
├── doc_kind_map.py         # код документа (docKindInfo.code) → подпапка
├── launcher.py             # обёртка: автоустановка pip-зависимостей при первом запуске
└── requirements.txt        # зависимости Python (mcp, requests)
```

## Подключение к Claude Code

Через плагин Claude Desktop — MCP регистрируется автоматически через
`.mcp.json` плагина. `launcher.py` при первом запуске сам ставит
pip-зависимости. Нужен только Python 3.10+ в системе.

После установки плагина — перезапуск Claude, проверить через `@zakupki-eis` в новом чате.
