# Структура жалобы / пояснений в УФАС

Каркас работает для всех трёх режимов скилла (A / C / D) с лёгкими адаптациями. Для режима C («пояснения против РНП») вместо «жалобы» — «письменные пояснения», просительная часть смещается в сторону «не включать в реестр».

---

## Полный каркас

```
ШАПКА

[Адресат]
В Управление Федеральной антимонопольной службы
по [региону]

[Заявитель]
От: [ИП / ООО], ИНН/КПП, юр. адрес, контактный телефон,
e-mail, представитель (если есть)

[Тип документа]
ЖАЛОБА на действия [заказчика] при проведении закупки № [рег. №]
(или: ПИСЬМЕННЫЕ ПОЯСНЕНИЯ по делу о включении в РНП по контракту № …
или: ЗАЯВЛЕНИЕ о нарушении антимонопольного законодательства
по ст. 16, 17 135-ФЗ)

I. СВЕДЕНИЯ О ЗАКУПКЕ

  1. Реестровый номер: 0000000000
  2. ИКЗ: 000000000000000000000000000000000
  3. Заказчик: …, ИНН …
  4. Способ определения поставщика: …
  5. ЭТП: …
  6. НМЦК: … ₽
  7. Цена контракта: … ₽ (если уже заключён)
  8. Реестровый номер контракта: … (если применимо)
  9. Дата заключения контракта: … (если применимо)

II. ОБСТОЯТЕЛЬСТВА

  Каждое обстоятельство — отдельным блоком в формате
  «обстоятельство → ↓ → вывод» (см. style-usenkova.md).

  Обстоятельство № 1
  [Факт со ссылкой на документ и пункт. Цитата если ключевая.]

  ↓

  [Вывод: правовая квалификация со ссылкой на норму.]

  Обстоятельство № 2
  …

III. ПРАВОВАЯ КВАЛИФИКАЦИЯ

  Обобщающий раздел. Перечислить нормы, на которые ссылается
  заявитель, и связать их с приведёнными обстоятельствами.

IV. ПРОСИТЕЛЬНАЯ ЧАСТЬ

  Прошу:
  1. Признать жалобу обоснованной.
  2. Выдать предписание об устранении нарушений (для типа A до торгов).
  3. Передать материалы для привлечения виновных к адм. ответственности.
  4. Не включать сведения об Исполнителе в РНП (для режима C).

V. ПРИЛОЖЕНИЯ

  Перечислить все документы, на которые ссылается жалоба.
  Каждый — с номером, датой, количеством листов.

[Дата] [Подпись + расшифровка]
```

---

## Различия по режимам

### Режим A — жалоба на документацию

**Адресат:** УФАС по месту нахождения заказчика (не по нашему месту нахождения).

**Срок:** до окончания срока подачи заявок (ч. 4 ст. 105 44-ФЗ).

**Просительная часть** — стандартная (признать обоснованной, выдать предписание, передать для адм. ответственности).

**Особенность:** перед подачей убедиться, что **у заявителя есть правоспособность участвовать** в этой закупке (с 2024 года ФАС не рассматривает жалобы от лиц, чьи права не могут быть нарушены — см. [chek-list.md](chek-list.md)).

---

### Режим C — пояснения против РНП

**Заголовок:** не «Жалоба», а «Письменные пояснения для [регион]ского УФАС России по вопросу о включении сведений в РНП по контракту № …».

**Адресат:** УФАС, в которое заказчик направил материалы (обычно по своему месту нахождения).

**Срок:** до заседания комиссии РНП. Подаётся параллельно с участием в заседании.

**Структура обстоятельств** — упор на хронологию открытых действий подрядчика и наличие спора по толкованию (см. [argumenty-rnp.md](argumenty-rnp.md)).

**Просительная часть:**
```
Прошу:
1. Не включать сведения об Исполнителе ([ИП ФИО / ООО ...])
   в Реестр недобросовестных поставщиков по контракту № ...
   от [дата] в связи с отсутствием в его действиях
   признаков недобросовестности, необходимых для применения
   указанной меры ответственности.
2. Принять во внимание прилагаемые материалы.
```

**Доп. секция перед просительной — рекомендуется:**
«Ключевые обстоятельства, подтверждающие добросовестность Исполнителя» — таблица из 5–6 обстоятельств, чем подтверждается, значение для УФАС. (Образец — Ставропольский кейс.)

---

### Режим D — антимонопольное заявление

**Заголовок:** «Заявление о нарушении антимонопольного законодательства (ст. 16, 17 ФЗ № 135 «О защите конкуренции»)».

**Адресат:** УФАС по региону. Можно одновременно — в центральный аппарат ФАС России для крупных дел (например, картельный кабинет).

**Срок:** не привязан к торгам, подаётся в любое время после выявления признаков сговора.

**Особенности структуры:**
- В разделе I добавить «Лица, в действиях которых усматриваются признаки нарушения» (заказчик + участники).
- В разделе II обстоятельства группировать: на стадии обоснования НМЦК → на стадии торгов → на стадии исполнения.
- В разделе IV дополнительно просить «возбудить дело о нарушении антимонопольного законодательства» (ст. 39, 44 135-ФЗ).

**Параллельно** — обращение в прокуратуру района (по нарушению ст. 22 44-ФЗ при обосновании НМЦК).

---

## Канал подачи

**Только электронно через ЕИС с усиленной квалифицированной электронной подписью (УКЭП).**

В жалобе обязательно указать:
- Контактные данные: e-mail (для уведомлений), телефон (для связи с комиссией УФАС)
- Способ получения копии решения: ЕИС / e-mail / почта

Бумажная подача — недопустима по ч. 6 ст. 105 44-ФЗ для всех жалоб с 2022 года.

---

## Объём

- Жалоба типа A на документацию: 4–8 страниц + приложения
- Пояснения против РНП: 6–12 страниц + приложения (стержневой документ кампании)
- Антимонопольное заявление: 8–15 страниц + развёрнутые приложения с расчётами и записями

**Не раздувать.** Каждое обстоятельство — одна страница максимум. Если есть, что добавить — отдельным обстоятельством.
